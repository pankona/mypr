# これは何をするかと言うと

* 自分にアサインされているプルリクエスト一覧をブラウザで表示します。

# Install

* `go get github.com/pankona/mypr`

# Usage

* 以下の環境変数を設定します。

  * MYPR_GITHUB_USERNAME
    * GitHub のユーザ名

  * MYPR_GITHUB_ORG
    * Organization 名。こちらの指定は任意。

  * 以下のコマンドを実行すると、自分にアサインされているプルリクエスト一覧が表示されます。
    * `$ mypr`
